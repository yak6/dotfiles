# yak6's dotfiles
My configuration files for programs I regularly use.

In this repository you can find:
- Configuration files for:
  - bash (shell)
  - alacritty (terminal)
  - vim/neovim (text editor)
  - tmux (terminal multiplexer)
  - i3 and i3blocks (window manager)

![i3-setup](https://github.com/user-attachments/assets/64a51627-6edc-4fa4-9cef-fd4733fe7ec5)

Requirements for i3 and i3blocks config:
- maim (for screenshots)
- pulseaudio (for audio volume)
- Iosevka Nerd Font (NerdFonts for icons on i3blocks)
- rofi (application runner)
- alacritty (terminal)
