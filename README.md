### Install plugins
Install vim-plug to manage plugins in vim:

```bash
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

In vim, type:

```
:PlugInstall
```

This will install all the plugins from .vimrc
